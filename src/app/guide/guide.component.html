🔐 Authentication Guide

This guide will show you how to authenticate with our app and test all secured APIs.

1️⃣ Open Swagger – Test APIs

Click the button below to open the interactive API documentation (Swagger):

🚀 Open Swagger API Tester

2️⃣ Use Test Credentials to Login

Call the /auth/login API using the following test credentials:

  • Email: testuserexample.com
  • Password: Test1234!

3️⃣ Check Your Email for the Code

You will receive a 6-digit code in your email inbox.

4️⃣ Validate the Code to Get Your Token

Call /auth/validate with the code you received to retrieve your JWT token.

5️⃣ Paste Your Token in Swagger

Click “Authorize” in the Swagger header and paste your token. This applies your token to all future requests.

⏱️ Note: The token is valid for 15 minutes. If you encounter a 401 error, simply repeat the login process.
